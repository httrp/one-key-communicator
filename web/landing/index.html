OKC One-Key-Communicator
How it works Features GitHub

Kommunizieren mit nur einer einzigen Taste

Kostenlos, quelloffen und radikal einfach. Für Menschen, die nicht mehr sprechen können — aber trotzdem alles zu sagen haben.

Jetzt starten

So funktioniert es

1

Öffnen

Öffne OKC im Browser auf deinem Tablet oder Handy. Kein Download nötig.

2

Raum erstellen

Erstelle einen Raum mit einem Tipp. Du bekommst einen QR-Code zum Teilen.

3

QR-Code scannen

Deine Familie scannt den QR-Code — sie sehen sofort, was du schreibst.

4

Kommunizieren

Drücke deine Taste, um Buchstaben auszuwählen. Der Rest passiert automatisch.

Eigenschaften

🎯 Eine Taste reicht

Alles funktioniert mit einem einzigen Tastendruck, Touch oder Sensor.

📱 Läuft überall

Handy, Tablet, Laptop, TV — jedes Gerät mit Browser funktioniert.

🔒 Privat

Kein Account, kein Tracking. Räume löschen sich automatisch.

🌍 Mehrsprachig

Deutsch, Englisch, Französisch, Spanisch und weitere Sprachen.

💬 Schnell-Sätze

Vordefinierte Sätze für häufige Situationen: Essen, Trinken, Hilfe.

🆓 Kostenlos & Open Source

Komplett kostenlos. Jeder kann OKC selbst betreiben.
