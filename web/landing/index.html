So wird kommuniziert

Schritt 1

Browser öffnen

One-Key läuft direkt im Browser — Handy, Tablet, Laptop. Keine App, kein Download.

Schritt 2

Raum erstellen

Ein PIN-gesicherter Raum wird automatisch angelegt. QR-Code erscheint sofort.

Schritt 3

QR-Code teilen

Bezugspersonen scannen den Code und sehen in Echtzeit, was geschrieben wird.

Schritt 4

Schreiben

Ein Tastendruck scrollt durch die Buchstaben, ein zweiter wählt aus.

Live-Demo

So sieht die Buchstabenauswahl aus. Die Markierung wandert automatisch — ein einziger Tastendruck wählt den aktuellen Buchstaben aus.

Eigenschaften

Was One-Key bietet

Eine Taste genügt Jeder Taster, Klick, Touch oder Sensor funktioniert. Kein spezielles Gerät notwendig.
Kein Account Kein Login, kein Tracking. Räume sind PIN-geschützt und werden nach 24h Inaktivität gelöscht.
Läuft überall Jedes Gerät mit aktuellem Browser. Kein App-Store, keine Installation, kein Update.
8 Sprachen Deutsch, Englisch, Französisch, Spanisch, Italienisch, Polnisch und weitere.
Schnell-Sätze Vorformulierte Sätze für häufige Situationen: Schmerz, Durst, Hilfe rufen, Ja/Nein.
Sprachausgabe Geschriebener Text wird auf Wunsch durch die Browser-TTS-Engine vorgelesen.
