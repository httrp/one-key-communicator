In 4 Schritten kommunizieren

Keine App-Installation, kein Account, keine Konfiguration. Einfach öffnen und loslegen.

1

Öffnen

Öffne One-Key Communicator im Browser auf Tablet oder Handy.

2

Raum erstellen

Ein Raum wird automatisch erstellt. Du bekommst einen QR-Code.

3

QR-Code teilen

Familie scannt den Code — sie sehen sofort, was du schreibst.

4

Kommunizieren

Drücke deine Taste, um Buchstaben auszuwählen. Fertig!
