    Wie funktioniert OKC?

    One-Key-Communicator ermöglicht Kommunikation mit nur einer Taste:

    1. Der Marker wandert automatisch über die Buchstaben
    2. Drücke eine beliebige Taste (oder tippe/klicke), um den markierten Buchstaben auszuwählen
    3. Der Runner scannt auch die Toolbar-Buttons (Sätze, Vorlesen, Teilen, etc.)
    4. Teile den QR-Code, damit andere mitlesen können

    Tipp: Die Geschwindigkeit kann in den Einstellungen angepasst werden.
