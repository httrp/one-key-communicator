# Beitragen zu One-Key

Vielen Dank für dein Interesse, zu One-Key beizutragen! Dieses Projekt ist für Menschen gedacht, die aufgrund eingeschränkter Mobilität nur mit einer einzigen Taste kommunizieren können. Jeder Beitrag, der das Tool verbessert, ist willkommen.

## Wie du helfen kannst

### 🐛 Fehler melden
Wenn du einen Fehler findest, öffne bitte ein [Issue auf GitHub](https://github.com/httrp/one-key-communicator/issues). Beschreibe:
- Was du erwartet hast
- Was stattdessen passiert ist
- Welches Gerät/Browser du verwendet hast

### 💡 Ideen & Feedback
Hast du Ideen, wie One-Key besser werden könnte? Teile sie gerne:
- Eröffne ein Issue mit dem Label "enhancement"
- Oder schreibe an: contact@one-key-communicator.com

### 🎛️ Hardware-Tipps
Besonders wertvoll sind **praktische Erfahrungen mit Hardware**:
- Welche Taster/Schalter funktionieren gut?
- Gibt es günstige Alternativen zu teuren Hilfsmitteln?
- Wie hast du One-Key in der Praxis eingerichtet?

Solche Erfahrungsberichte können in den [Hardware-Guide](/hardware) einfließen und anderen helfen.

### 🌍 Übersetzungen
One-Key unterstützt mehrere Sprachen. Wenn du helfen möchtest, eine Sprache zu verbessern oder hinzuzufügen, eröffne ein Issue.

### 💻 Code-Beiträge
Pull Requests sind willkommen! Bitte beachte:

1. **Fork & Branch**: Forke das Repository und erstelle einen Feature-Branch
2. **Kleine Schritte**: Lieber mehrere kleine PRs als ein riesiger
3. **Teste lokal**: Stelle sicher, dass alles funktioniert
4. **Beschreibe die Änderung**: Was hast du geändert und warum?

```bash
# Projekt lokal starten
go run ./cmd/okc

# Oder mit make
make run
```

## Code of Conduct

Bitte sei respektvoll und hilfsbereit. Dieses Projekt richtet sich an Menschen mit Behinderungen und ihre Angehörigen. Konstruktives Feedback ist willkommen, Negativität nicht.

## Lizenz

Durch deinen Beitrag stimmst du zu, dass er unter der [GPLv3-Lizenz](LICENSE) veröffentlicht wird.
